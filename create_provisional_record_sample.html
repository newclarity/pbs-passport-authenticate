This page has the code on it you'd need for a "thankyou" donate page. On page load, it attempts to create a provisional record with the values in the HTML below.

If the transaction ID (aka the last 6 digits of the 'confirmation code' in your Luminate Online TY email) matches a transaction made in the last 24 hours for the email address below, the system will check to see if the email is already in use in the station's Membership Vault. If not, it will create the provisional record and display an activation link. If the email already exists and has not activated, it will display an activation link for the current Membership Vault account. If the email already exists and has previously activated, it will display a login link.
